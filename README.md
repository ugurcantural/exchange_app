# döviz kurları

**Flutter** kullanarak geliştirmiş olduğum döviz uygulaması. Güncel döviz kurlarını görüntüleyebilir, TL cinsinden karşılığını sorgulayabilirsiniz. Kendi cüzdanınızı oluşturup, kaydedebilirsiniz.
